# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release of SourceScribe
- Multi-LLM support (Anthropic Claude, OpenAI GPT, Ollama)
- Real-time file watching with configurable debouncing
- Mermaid.js diagram generation (architecture, class, flow, sequence, ER diagrams)
- Multi-language code parsing support (Python, JavaScript, TypeScript, Java, Go, Rust, C/C++, and more)
- Pydantic-based configuration system with YAML support
- CLI interface with commands: generate, watch, init, validate, info
- Code analysis and structure extraction
- API endpoint detection and documentation
- Comprehensive test suite
- Cross-platform support (macOS, Linux, Windows)
- Python 3.7+ compatibility

### Features
- Automatic project overview generation
- Individual file documentation
- Architecture documentation with diagrams
- API documentation generation
- Dependency analysis
- Code metrics (optional)
- Configurable documentation styles (minimal, normal, detailed)
- Incremental documentation updates
- Rich console output with progress indicators

### Documentation
- Comprehensive README with examples
- Contributing guidelines
- Usage examples
- API documentation
- Configuration examples
